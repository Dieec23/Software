Software Repository for colaboration between Jared Stewart and Scott Visser

general git steps:
1. import repository to pwd
	git clone https://github.com/UNTDDCI/Software.git

2. edit contents

3. fetch any new/remote changes commited since you have started working
	git fetch

4. merge remote changes
	git merge

5. add/commit your changes

6. push commit to github remote
	git push -u origin master
